# Old Stuff
A folder for old utility python scripts. I might categorize and compres these helper functions later but I don't feel like dealing with them.
They are imprisoned in a directory (NOT A package) so their errors aren't seen by the django app when it boots up.
Probably some useful stuff in here that might be refactored and applied somewhere later.

Personally I don't like a whole app dedicated to utils. I feel like it is better to keep the methods next to the
items they are used the most by and then just have everyone else reference them in the distance.
